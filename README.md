Student Career Sentiment Analyzer

Project Overview

This project analyzes student reviews and classifies them into Positive, Negative, and Neutral sentiments using Natural Language Processing (NLP).

Objectives

* Analyze student opinions
* Perform sentiment classification
* Visualize sentiment distribution
* Generate insights from textual data

Technologies Used

* Python
* Pandas
* TextBlob
* Matplotlib
* WordCloud

Project Workflow

1. Collect student reviews
2. Clean and preprocess text data
3. Apply sentiment analysis
4. Classify reviews
5. Visualize results
6. Generate word cloud

Features

* Positive/Negative/Neutral Classification
* Pie Chart Visualization
* Bar Chart Visualization
* Word Cloud Generation

Files

* sentiment_analysis.py
* count.py
* pie.py
* bar.py
* generate_wordcloud.py
* student_reviews.csv
* sentiment_results.csv

Results

The project provides insights into student opinions regarding coding, internships, placements, and learning experiences.

Future Improvements

* Use larger datasets
* Apply machine learning models
* Real-time social media sentiment tracking
